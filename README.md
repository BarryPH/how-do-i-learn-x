# how-do-i-learn-x
> CRUD website to share the best available resources for learning topic 'x'. Created to learn React.js & Graphql

## Installing / Getting started

```shell
# Install dependencies
npm i

# Start
npm run dev
```

## Developing

### Built With
	* react.js
	* graphql
	* express.js
	* mongodb/mongoose
	* styled-components

### Prerequisites
	* node
	* npm
	* mongodb
